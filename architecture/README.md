# AWS CI/CD Pipeline Project

## 🏗️ Project Overview
A complete CI/CD pipeline built on AWS to automatically build, test, and deploy a Java web application.

## 🏛️ Architecture
Bitbucket → AWS CodeBuild → AWS Elastic Beanstalk
Connected via AWS CodePipeline

## ☁️ AWS Services Used
- AWS Elastic Beanstalk
- AWS RDS (MySQL)
- AWS CodeBuild
- AWS CodePipeline
- Amazon S3
- IAM

## 📁 Project Structure
- 01_Beanstalk - Elastic Beanstalk setup
- 02_RDS - Database setup
- 03_Bitbucket - Source control
- 04_CodeBuild - Build configuration
- 05_CodePipeline - Pipeline setup
- 06_Final_Verification - Testing
